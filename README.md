MAGNATE XMD

A powerful WhatsApp Multi-Device bot built with Baileys. Fast, stable, customizable, and packed with useful features for group management, fun commands, downloads, and automation.

Features

- Multi-Device Support
- Auto Reconnect
- Group Management Commands
- Download Commands
- Fun Commands
- Owner Commands
- Anti-Link System
- Anti-Spam Protection
- Easy Deployment

Requirements

- Node.js 18 or higher
- Git
- WhatsApp Account
- Stable Internet Connection

Installation

git clone https://github.com/your-username/Magnate-XMD.git
cd Magnate-XMD
npm install

Running the Bot

npm start

Or:

node index.js

Deploy to Render

1. Fork this repository.
2. Create a Render account.
3. Click the Deploy button below.
4. Connect your GitHub account.
5. Select the forked repository.
6. Add required environment variables.
7. Deploy and wait for the build to finish.

Deploy to Railway

1. Fork the repository.
2. Create a Railway account.
3. Create a new project.
4. Select "Deploy from GitHub Repo".
5. Choose your forked repository.
6. Add environment variables.
7. Deploy the project.

Environment Variables

SESSION_ID=your_session_id
PREFIX=.
OWNER_NUMBER=255XXXXXXXXX
